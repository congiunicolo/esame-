# OttoCN - Sito Web Bar

## Descrizione
Questo progetto rappresenta il sito web istituzionale del Bar "OttoCN" situato a Milano. Il sito è composto da 5 pagine HTML statiche collegate tra loro e stilizzate con un foglio di stile CSS personalizzato.

## Struttura del Progetto
- `index.html`: Home page.
- `menu.html`: Pagina dei servizi/menu.
- `dove-siamo.html`: Dove siamo.
- `contatti.html`: Informazioni di contatto.
- `prenotazioni.html`: Pagina con link all'applicativo di prenotazione.
- `style.css`: Stili globali del sito.
- `sitemap.md`: Documentazione architetturale del sito.

## Installazione e Utilizzo
1. Scaricare o clonare la repository.
2. Aprire il file `index.html` in un qualsiasi browser moderno.
3. Navigare tra le sezioni utilizzando la barra di navigazione superiore.
